# Troubleshooting

## Agent Not Responding

- Verify agent is assigned to phone number
- Check remaining plan minutes
- Confirm campaign status is **Active**

## Agent Repeats Questions

- Add more phrase variations to branching logic
- Simplify question phrasing

## Incorrect Answers

- Update knowledge base documents
- Add to prompt: "Only answer from your knowledge base"

## Language Not Switching

- Enable **Auto-Detect Language**
- Confirm language in **Additional Languages**

## Contacts Not Syncing from Zoho

- Enable **Bidirectional Sync**
- Verify phone numbers in E.164 format
- Click **Sync Now** in Integrations

## Low Connection Rates

- Call 10am-12pm or 3pm-5pm on weekdays
- Rotate caller IDs if spam-flagged
- Use local numbers for higher pickup

## Outcomes Not Logging to CRM

- Check **Bidirectional Sync** enabled
- Verify field names match exactly in Zoho
- Use long-text fields for transcripts

## Out of Minutes

- Upgrade plan or top up usage
- Check **Billing** dashboard

Contact [support](/support/overview) for help.
