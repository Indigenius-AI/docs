# Web Call

Learn to make a **web call** to your Indigenius agent directly from the browser.

---

## Overview

This guide shows you how to make a web call to your **Indigenius agent**.  
You can create an assistant using either:

- The **Web SDK**  
- The **Indigenius Dashboard**

---

## Environment Setup (v0)

1. Go to **Settings → Environment Variables** in your app.  
2. Create a new environment variable:  

```bash
NEXT_PUBLIC_INDIGENIUS_API_KEY=your-public-api-key
````

3. Add your **public API key** from the dashboard.

---

Install the SDK with your preferred package manager:

<CodeGroup>

## Quick Start

\<Tabs items={\["HTML", "React"]}>

<TabItem value="HTML">

```html
<indigenius-convai agent-id="your-agent-id"></indigenius-convai>

<script type="module">
  import { CdialWidget } from "indigenius-ai-widget";

  const widget = new CdialWidget({
    agentId: "your-agent-id",
    mountUI: true,
  });

  widget.init();
</script>
```

</TabItem>

<TabItem value="React">

```tsx
import { useEffect } from "react";
import { CdialWidget } from "indigenius-ai-widget";

export default function App() {
  useEffect(() => {
    const widget = new CdialWidget({
      agentId: "your-agent-id",
      mountUI: true,
    });
    widget.init();

    return () => widget.destroy();
  }, []);

  return <div>Your React App</div>;
}
```

</TabItem>

</Tabs>

## Call Control

Start or end a call programmatically:

```js
widget.toggleCall();
```

---

## Event Listeners

```js
widget.on("dialing", () => console.log("Dialing..."));
widget.on("connected", () => console.log("Connected"));
widget.on("ended", () => console.log("Call ended"));
widget.on("error", (err) => console.error("Error:", err));
```

---

## Widget Options

| Option    | Type      | Required | Default | Description                                                                                             |
| --------- | --------- | -------- | ------- | ------------------------------------------------------------------------------------------------------- |
| `agentId` | `string`  | ✅ Yes    | —       | The unique Agent ID for your AI agent.                                                                  |
| `mountUI` | `boolean` | ❌ No     | `false` | If `true`, auto-mounts the floating UI. If `false`, no UI is shown and you control it programmatically. |

---

## Available Events

| Event       | Payload | Description                                                       |
| ----------- | ------- | ----------------------------------------------------------------- |
| `dialing`   | `void`  | Fired when the widget is trying to connect to the AI agent.       |
| `connected` | `void`  | Fired once the connection with the AI agent is successfully made. |
| `ended`     | `void`  | Fired when the call ends or is terminated.                        |
| `error`     | `Error` | Fired when an error occurs in the widget.                         |
