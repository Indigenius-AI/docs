# Language Preferences

Indigenius agents are **multilingual by design**. Agents can detect the caller's language automatically, switch mid-conversation, and deliver natural responses in the caller's preferred language.

---

## Supported Languages

Indigenius currently supports:

| Language | Code | Notes |
| --- | --- | --- |
| English | `en` | Default. Multiple accents supported |
| Yoruba | `yo` | Nigerian regional language |
| Igbo | `ig` | Nigerian regional language |
| Hausa | `ha` | Nigerian regional language |
| Pidgin English | `pcm` | Nigerian Pidgin |
| French | `fr` | Standard and West African French |
| Spanish | `es` | Latin American and European Spanish |

<Note>
  Additional languages are added regularly. Contact [support](/support/overview) to request a specific language.
</Note>

---

## Configuring Language Settings

1. Go to **Agents →** open your agent **→ Settings** tab
2. Under **Language**, set your **Primary Language**
3. Toggle **Auto-Detect Language** to allow the agent to switch based on the caller's speech
4. Add **Additional Languages** from the dropdown (up to 5 per agent)
5. Set a **Fallback Language** — used when detection confidence is low
6. Click **Save**

---

## Language Flow Logic

When **Auto-Detect Language** is enabled:

```text
Caller speaks → STT detects language → Agent responds in same language
                              ↓
               (If confidence < 80%) → Use Fallback Language
```

### Explicit Language Switch Prompt

You can also instruct the agent to ask the caller for their preferred language at the start of the call:

```text
Begin every call by saying:
"Welcome to Acme Solar. You can speak to me in English, Yoruba, or Hausa. Which do you prefer?"
Then respond in the language the caller chooses for the rest of the call.
```

---

## Language-Specific Greetings

You can define different opening greetings per language in your system prompt:

```text
If the caller speaks Yoruba, greet them with: "E kaabo! Mo jẹ Amara, oluranlowo AI fun Acme Solar."
If the caller speaks Hausa, greet them with: "Barka da zuwa! Ni Amara ne, mataimakiyar AI ta Acme Solar."
Otherwise, greet in English.
```

---

## Best Practices

- **Always set a fallback language.** Without one, detection failures may cause the agent to default to silence.
- **Test each language explicitly.** Run a test call in each configured language. See [Testing Checklist](/testing/checklist).
- **Keep scripts culturally appropriate.** Direct translations can miss local nuance. Review scripts with native speakers before launch.
- **Limit to 3 languages for complex workflows.** More languages with deep branching logic can become difficult to maintain.

---

## Troubleshooting

**Agent responded in the wrong language.** Check that Auto-Detect is enabled and the language is listed under Additional Languages. If confidence is low, the fallback language will be used.

**Yoruba/Hausa responses sound unnatural.** Add voice tuning notes to your prompt. For example: _"When speaking Yoruba, use a warm, community-focused tone."_
